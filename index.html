Hello everyone, I have used the Protest,kode,micro,oswald font here.

The Protest Revolution font has the following description:

  • Octavio Pardo designed the Protest Revolution font.
  • Originally, it was a paint brush effect used as an experiment, but it became a real font after that.
  • Web pages with educational content and children's websites mostly use the Protest Revolution font in HTML.
  • A Free Font License is included with this font. The printable files can be used in a variety of products & projects, be it print or digital, commercial or otherwise.

Here is a step-by-step explanation of how I used Protest Revolution font on my website:

      1. Fonts.google.com provided this Protest Revolution font for me
      2. Once I clicked on it, I was able to open Protest Revolution font.
      3. On the top-right corner of the font window, I found a button called Get font.
      4. On the right-side of the webpage, following the Get font button, you can find two buttons titled Get embed code and Download all.
      5. In the new window that opened, I clicked on the get embed code button, and I selected web from the options.
      6. There are html code and CSS classes in the web that I selected.
      7. The head tag included the HTML code, and my style sheet included the CSS classes.
